# INSTRUCCIONES PARA DESPLEGAR EN VERCEL

## Problema actual:
Error 404: NOT_FOUND - Vercel no encuentra el index.html

## Solución:

### Método 1: Configurar Root Directory en Vercel (RECOMENDADO)
1. Ve a tu proyecto en Vercel (https://vercel.com/dashboard)
2. Click en tu proyecto
3. Ve a Settings
4. En la sección 'General' busca 'Root Directory'
5. Si subiste toda la carpeta 'incargo', pon: logistica-1.0.0
6. Si subiste solo 'logistica-1.0.0', déjalo en blanco o pon: ./
7. Haz click en 'Save'
8. Ve a 'Deployments' y haz un nuevo deploy

### Método 2: Subir solo el contenido de logistica-1.0.0
1. En lugar de subir la carpeta 'incargo' o 'logistica-1.0.0'
2. Sube directamente los archivos que están DENTRO de logistica-1.0.0:
   - index.html
   - about.html
   - service.html
   - (todos los demás archivos HTML)
   - carpetas: css/, js/, lib/, img/
   - vercel.json

### Método 3: Usar Vercel CLI
Desde la terminal PowerShell, en la carpeta logistica-1.0.0:

1. Instala Vercel CLI (solo la primera vez):
   npm install -g vercel

2. Navega a la carpeta correcta:
   cd c:\Users\desar\Documents\incargo\logistica-1.0.0

3. Despliega:
   vercel

4. Sigue las instrucciones en pantalla

## Archivo vercel.json creado
Ya he creado el archivo vercel.json con la configuración correcta.
Solo necesitas volver a desplegar siguiendo uno de los métodos anteriores.
